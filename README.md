<h2>Overview</h2>
<p>To-do-list is a straightforward, Node.js-powered application built with Express that demonstrates essential RESTful API principles for managing tasks. It allows users to add and delete tasks through dedicated routes, providing a practical example of server-side CRUD operations.</p>
